# study Go

### Get Start

1. clone project
    ```
    https://github.com/kick0131/studyGo.git
    ```
1. go run
    ```
    go run ./main/main.go
    ```

- Don't set environment

    | key | value |
    | ---- | ---- |
    | GOROOT | installed go path |
    | GOPATH | use default working directory |
    | PATH | execute go path(go.bin,go.exe) |
